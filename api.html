The image (im) accessor

The image accessor enables to easily subset the image container.

class spatial_data.im.image.ImageAccessor(xarray_obj)

The image accessor enables fast indexing and preprocesses image.data

add_channel(channels: Union[str, list], array: numpy.ndarray)

Adds channel(s) to an existing image container.

colorize(colors: List[str] = ['C0', 'C1', 'C2', 'C3'], background: str = 'black', normalize: bool = True, merge=True) → xarray.core.dataset.Dataset

Colorizes a stack of images.

Parameters

  • colors (List[str]) – A list of strings that denote the color of each channel.

  • background (float) – Background color of the colorized image.

  • normalize (bool) – Normalizes the image prior to colorizing it.

  • merge (True) – Merge the channel dimension.

Returns

The image container with the colorized image stored in Layers.PLOT.

Return type

xr.Dataset

The plot (pl) accessor

The plotting accessor provides several plotting functions.

class spatial_data.pl.plot.PlotAccessor(xarray_obj)

Adds plotting functions to the image container.

add_box(xlim: List[int], ylim: List[int], color: str = 'w', linewidth: float = 2, ax=None)

Adds a box to the current plot.

Parameters

  • xlim (List[int]) – The x-bounds of the box [xstart, xstop].

  • ylim (List[int]) – The y-bounds of the box [ymin, ymax].

  • ax (matplotlib.axes) – Matplotlib axis to plot on (default: None)

Returns

The image container.

Return type

xr.Dataset

annotate(highlight: list = [], text_kwargs: dict = {'color': 'w', 'fontsize': 12}, highlight_kwargs: dict = {'color': 'w', 'fontsize': 16, 'fontweight': 'bold'}, bbox: Optional[List] = None, ax=None) → xarray.core.dataset.Dataset

Annotates cells with their respective number.

Parameters

  • highlight (List[int]) – A list with cell ids which are highlighted in the plot.

  • text_kwargs (dict) – Keyword arguments that are passed to matplotlib’s text function.

  • hightlight_kwargs (dict) – Similar to text_kwargs but specifically for the cell ids that are passed via highlight.

  • ax (matplotlib.Axes) – Matplotlib axis.

Returns

The image container.

Return type

xr.Dataset

The segmentation (se) accessor

The segmentation accessor contains methods that interrogate with the segmentaton mask.

class spatial_data.se.segmentation.SegmentationAccessor(xarray_obj)

Handles everything that relates to the provided segmentation.

add_segmentation(segmentation: numpy.ndarray, copy: bool = True) → numpy.ndarray

Adds a segmentation mask (_segmentation) and an obs (_obs) field to the xarray dataset.

Parameters

  • segmentation (np.ndarray) – A segmentation mask, i.e. a np.ndarray with image.shape = (n, x, y), that indicates the location of each cell.

  • copy (bool) – If true the segmentation mask is copied.

Returns

The amended xarray.

Return type

xr.Dataset
